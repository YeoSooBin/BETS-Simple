Hello Friend!

It is possible to activate filters by inputing flags into search boxes, but most layman users would not be able to do so easily. What we have done is to pre-program such a filtered Search using Google's Customisable Search Engine(CSE), and presenting it in this page.

For the purpose of this prototype, we have limited the search to 30 web domains that focus on future trends. Use this search function to get a feel of the "hit-rate"! You can also request for addition and deletion of web-domains to be covered by writing to yeosoobin@gmail.com.

Use Navbar at the top to check out the Editor module and NotebookLM to help generate Products for your audience!
